# D&D Music Player for real DM's 🧙‍♂️

Атмосферный музыкальный плеер для настольных RPG (Dungeons & Dragons и не только), с быстрым доступом к категориям треков через **drumpad-интерфейс**
Создан для мастеров, которым надо **мгновенно переключать атмосферу**: бой, таверна, лес, эпик - без копания в папках как идиот :P

---

## Возможности

* **Drumpad категорий**

  * Быстрый запуск музыки по категориям
  * Визуальные кнопки с цветами и иконками
  * Отображение количества треков

* **Случайное воспроизведение**

  * Клик по категории → случайный трек
  * Идеально для DnD-сцен

* **Горячие клавиши**

  * `1–9` → запуск категорий
  * `Space` → play/pause
  * `↑ / ↓` → громкость

* **Управление треками**

  * Добавление/удаление в категории
  * Отдельный список всех файлов

* **Режимы воспроизведения**

  * Loop (зацикливание)
  * Авто-переход к следующему треку

* **Кастомизация**

  * Цвет категории
  * Иконки (эмодзи)
  * Названия и hotkeys в json

## Установка

### 1. Склонируйте репо

```bash
git clone git@github.com:gotogrub/DnD-Music-Player.git
cd dnd-music-player
```

### 2. Установить зависимости

```bash
pip install pygame PyQt5
```

## Запуск

```bash
python player.py
```

## Как пользоваться

1. Запусти приложение
2. Закинь музыку в папку `music/`
3. Добавь категории (кнопка `+`)
4. Добавь треки в категории
5. Жми на кнопку категории — и погнали

## ⌨️ Горячие клавиши

| Клавиша | Действие         |
| ------- | ---------------- |
| 1–9     | Запуск категории |
| Space   | Play / Pause     |
| ↑       | Громкость +      |
| ↓       | Громкость -      |

## Настройки

Все сохраняется в:

```
player_settings.json
```

Пример:

```json
{
  "categories": {
    "Битва": {
      "tracks": ["battle1.mp3"],
      "hotkey": "1",
      "icon": "swords",
      "color": "#8b0000"
    }
  }
}
```

## Поддерживаемые форматы

* `.mp3`
* `.wav`
* `.ogg`
* `.flac`

## Возможные проблемы

**Музыка не играет**

* Проверь, что файлы лежат в `music/`
* Убедись, что формат поддерживается

**Нет звука**

* Проверь систему (да, серьёзно, бывает…)

**Лаги / баги**

* pygame иногда тупит
